01 Elige tu método

Outlook ofrece tres formas de usar plantillas HTML. La más sencilla y recomendada es insertar el HTML como archivo al redactar.

02 Método recomendado · Insertar como texto

01

Descarga el template

Click en ↓ Descargar HTML arriba. Guarda el archivo en una carpeta accesible — recomiendo Documentos / Plantillas Correo.

02

Abre Outlook y crea un correo nuevo

En Outlook Desktop: Inicio → Nuevo correo electrónico (o Ctrl+N). Asegúrate de que el formato del mensaje sea HTML: Formato de texto → HTML.

03

Inserta el archivo como texto

En la ventana del correo: Insertar → Adjuntar archivo → flecha del botón → Insertar como texto.

📧  Sin título — Mensaje (HTML)
Archivo Mensaje Insertar Opciones Formato de texto Revisar
📎▾ Adjuntar archivo ▾ → Insertar como texto ⌘+I (Mac) · Alt+N+I (Win)

Selecciona el HTML descargado y se pegará formateado en el cuerpo del correo.

04

Personaliza el contenido

Edita directamente sobre el correo — los textos son editables. Cambia el nombre del proyecto, fechas, entregables, archivos, etc. Outlook mantiene el formato.

  • Headline · "Aurora · Fase 02" → tu proyecto
  • Eyebrow · "Entrega · Proyecto" → "Contrato", "Propuesta", "Avance semanal"…
  • Entregables · ajusta título + descripción
  • Archivos · cambia nombres y tamaños (esto es visual; adjunta los archivos reales aparte)
  • Botones CTA · cambia el link href del "Ver en GitHub" antes de pegar, o ignóralos
05

Adjunta archivos reales y envía

Los archivos del cuerpo son visuales. Para envío real, adjunta los archivos como adjuntos normales (Insertar → Adjuntar archivo).

03 Plantilla reutilizable (.oft)

Si vas a enviar este formato seguido, guárdalo como plantilla de Outlook:

  1. Inserta el HTML siguiendo los pasos arriba.
  2. Personaliza los campos genéricos (deja vacíos los que cambian).
  3. Archivo → Guardar como → tipo: Plantilla de Outlook (*.oft).
  4. Para usarla después: Nuevos elementos → Más elementos → Elegir formulario → Plantillas de usuario en el sistema de archivos.

04 Outlook Web / Outlook 365

La opción "Insertar como texto" no existe en la versión web. Usa este flujo alternativo:

  1. Abre el HTML descargado en un navegador (doble-click).
  2. Selecciona todo el contenido del correo (Ctrl+A o ⌘A).
  3. Copia (Ctrl+C / ⌘C).
  4. En Outlook Web, redacta un correo nuevo y pega (Ctrl+V / ⌘V).
  5. El layout, colores y logo se mantienen porque está todo en estilos inline.
▍ Tip

Para crear una "plantilla rápida" en Outlook Web 365 → Configuración → Correo → Redactar y responder → Mis plantillas → pega ahí el HTML pegado. Después insertas con un click.

▍ Pro tips · envío

  • Imagen del logo · el bindrune está embebido en base64 en el HTML — viaja contigo, no requiere internet ni se rompe en clientes restrictivos.
  • Modo oscuro · el template dark respeta su paleta original. El light a veces se "invierte" en clientes con dark mode forzado (Apple Mail, Outlook iOS). Es comportamiento del cliente, no del HTML.
  • Botones · cambia el href de "Ver en GitHub" a tu repo/URL real antes de guardar como plantilla.
  • Fuentes · usa Segoe UI en Outlook Windows, Helvetica en Mac. Si quieres exactamente Outfit, hay que usar @import de Google Fonts — pero algunos clientes lo strippean. Las web-safe fonts incluidas se ven 95% iguales al diseño.
  • Asunto · usa formato [Aurora · F02] Entrega de sprint · 24/05 para que el cliente identifique el proyecto rápido.
  • Tamaño · cada template pesa ~32 KB — perfecto para emails (Gmail trunca correos > 102 KB).

05 Personalizar el código (avanzado)

Si quieres modificar el HTML directamente — cambiar paleta, fuentes, secciones enteras — abre el archivo en cualquier editor de texto. Las partes customizables están bien marcadas:

<!-- Personalizar acento -->
background:#1f8a4d; /* verde light */
background:#4ade80; /* verde dark */

<!-- Cambiar fuente del headline -->
font-family:'Segoe UI',Calibri,Arial,...

<!-- Reemplazar imagen del logo -->
<img src="data:image/png;base64,...">
→ subir nuevo PNG a base64 (e.g. base64.guru)
